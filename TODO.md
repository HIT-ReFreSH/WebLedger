## 描述


创建一个现代的 React 管理后台作为本项目 WebLedger 的参考实现。这将作为开发者在构建前端应用程序时的示例。

**需要实现的功能：**

- 带有摘要统计信息的仪表盘概览
- 具有过滤和排序功能的条目列表
- 带验证的条目创建表单
- 分类管理页面
- 类型管理页面
- 响应式设计（移动端友好）

**技术栈：**

- React 18+ 与 TypeScript
- Vite 作为构建工具
- Yarn PnP 或 pnpm
- TanStack Router (React Router v6+) 用于路由
- TanStack Query 用于数据获取
- Tailwind CSS 或 Ant Design 用于 UI
- Zod 用于验证

## ⚠️ 重要要求

- 使用 **Vite**（不是 Create React App）
- 使用 **TypeScript**（不是 JavaScript）
- 使用 **yarn PnP** 或 **pnpm**（不是 npm）

## 验收标准 / 验收标准

- 项目使用 Vite + React + TypeScript 初始化
- 使用 yarn PnP 或 pnpm（不使用 npm）
- 所有组件均使用 TypeScript 编写（不包含.jsx 文件）
- 实现所有必需页面
- 完全响应式设计
- 错误处理和加载状态
- 代码注释良好并遵循最佳实践
- 带有设置说明的 README
